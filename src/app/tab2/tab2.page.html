How to use this app

Map View Information:
The map view in this application is created so that any user may view the shelters on the map, while also providing the user's location in relation to the shelters.
Shelter View Information:
The shelter view in this application is created so that any user may view a list of available shelters. A user may chose to select a specific shelter and gain some more information regarding the shelter's availability and supply levels.
How to Log In (Data Entry Only):
If you are a member of the Emergency Disaster System team, your log-in information should have been provided. If you forget your log-in information or are unable to log-in, please contact an administrator so that information may be recovered. Please log in so that you may continue completing your task.
